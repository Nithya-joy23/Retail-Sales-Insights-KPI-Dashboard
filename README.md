**Retail Sales Insights Analysis**
📌 Project Overview

This project analyzes retail sales data to identify trends, key performance indicators (KPIs), and business insights. The objective is to understand sales performance across categories, regions, and time periods to support data-driven decision making.

The project uses Python for data analysis and visualization to discover meaningful insights from retail transaction data.

🎯 Objectives
Analyze overall sales performance
Identify top-performing product categories
Evaluate regional profit distribution
Understand monthly sales trends
Identify best-selling products
Generate business insights using data visualization
🛠 Tools and Technologies
Python
Pandas
Matplotlib
Seaborn
Jupyter Notebook / VS Code
